    • Boxes

      A box is a place you can store something for as long as you'd like. You can use as many boxes as you like by giving each box a name. If you put something new in the box, it replaces anything already inside. Each script has its own boxes. You can use a box by putting its name in square brackets.

      [this is a box]

    • Notes

      A note is a line of text that koala ignores. Use it to explain what the script is supposed to do. This will help others and even yourself, should you forget, to understand your script. Notes are created by a double hypen.

      -- this is an example

    • Numbers

      A number is a set of digits with no space inbetween. You may optionally use a decimal point.

      123 45.0 -67 8.9

    • Script

      A script is a recipe for how a computer does something. A program is run by following one or many scripts. There are several scripts that are builtin to koala. A script can run any other script just by writing its name followed by any information you want to give it.

      say "hello"

    • Switch

      A switch has only two states. You can set it on or off, yes or no, true or false. Switches are useful when you want to make a decision.

      on yes true off no false

    • Text

      Text is created by writing between either quotes or double quotes. If you want to type a quote inside the text, put a backslash behind it.

      "this is an example" 'so is this'
